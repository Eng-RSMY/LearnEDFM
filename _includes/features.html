Features

LearnEDFM describes the coupled processes connected to fluid flow and transport in the fractured subsurface. We use the embedded discrete fracture model leading to a system of partial differential equations and solve these systems with MATLAB. LearnEDFM comes with the following key features:

  • 2D vectorized MATLAB implementation

  • Single phase darcy flow

  • Tracer transport (advection + diffusion + dispersion)

  • Heat transport (advection + diffusion)

  • Comprehensive user manual

Whether you're a student getting a first glimpse modeling naturally fractured reservoirs or a professional researcher looking for a new simulation tool LearnEDFM might be the perfect starting point for you!
